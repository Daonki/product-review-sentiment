# Product Review Sentiment Analysis (상품 리뷰 감성 분류)

📌 프로젝트 소개  
온라인 상품 리뷰 텍스트 데이터를 활용하여  
리뷰의 감정을 **긍정 / 부정 및 다중 감성(Multi-label)** 으로 분류하는  
머신러닝 기반 감성 분석 모델을 설계·분석한 프로젝트입니다.

텍스트 전처리부터 특징 벡터화, 모델 학습, 성능 평가 및 결과 해석까지  
자연어 처리(NLP) 기반 데이터 분석의 전 과정을 경험했습니다.

---

🛠 사용 기술
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

---

⚙️ 주요 기능
- 상품 리뷰 텍스트 데이터 전처리
- 감성 분류 모델 학습 및 예측
- 다중 감성(Multi-label) 분류 실험
- 예측 결과 시각화 및 오분류 사례 분석

---

👩‍💻 담당 역할
- 리뷰 텍스트 데이터 전처리 및 분석
- 감성 분류 모델 설계 및 학습
- 학습/검증 데이터 구성 및 성능 평가
- 시각화를 통한 결과 분석 및 해석

---

📈 결과
- 긍정/부정 감성 분류 모델 구축
- 속성 기반 감성 분류 및 다중 감성 분류 실험 수행
- 전처리 및 벡터화 방식에 따른 모델 성능 차이 확인

---

📂 프로젝트 구성
- `Step1_상품_리뷰_감성_분류.ipynb` : 기본 감성 분류 모델 구축
- `Step2_속성_검출(Multi_label)_분류.ipynb` : 속성 기반 다중 감성 분류
- `Step3_속성_기반_감정_분류.ipynb` : 속성과 감성을 결합한 분류 실험
